# Aid Worker Security Analysis Project

## Overview
This project analyzes global security incidents affecting humanitarian aid workers from 1997-2023. It provides interactive visualizations and detailed analysis of attack patterns, victim demographics, and temporal trends to inform security protocols and risk management strategies.

## Features
- Interactive global incident mapping
- Temporal trend analysis
- Attack pattern visualization
- Demographic breakdown
- Risk factor analysis
- Comprehensive recommendations

## Data Sources
Primary data sourced from the Aid Worker Security Database (AWSD), maintained by Humanitarian Outcomes. Dataset includes verified incidents of:
- Direct attacks
- Collateral damage
- Security incidents
- Casualties and kidnappings

- Python 3.8+

  - Plotly

## Usage
The project is structured as a Quarto website with interactive visualizations. Each page focuses on specific aspects of aid worker security:

- **Attack Characteristics**: Analysis of attack types and patterns
- **Attack Trends**: Temporal and geographic trend analysis
- **Victim Demographics**: Demographic breakdown of affected personnel
- **Conclusions**: Key findings and recommendations

## Interactive Features
- Click-and-drag globe rotation
- Year-based filtering
- Attack type selection
- Demographic filtering
- Zoom capabilities
- Hover information

## Key Visualizations
